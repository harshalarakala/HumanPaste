Having trouble opening (Gatekeeper)?
Install from Terminal after downloading the DMG: 
DMG=~/Downloads/HumanPaste.dmg
MNT=$(hdiutil attach -nobrowse -noverify -noautoopen "$DMG" | awk '/Volumes/{print $3}')
cp -R "$MNT/HumanPaste.app" /Applications/
hdiutil detach "$MNT"
xattr -dr com.apple.quarantine "/Applications/HumanPaste.app"
codesign --force --deep --sign - "/Applications/HumanPaste.app"
open "/Applications/HumanPaste.app"
Alternatively: right‑click the app in Finder → Open → Open.
